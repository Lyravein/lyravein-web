Denia

AI companion untuk Discord server kamu. Chat cerdas, web search real-time, dan banyak lagi — semua dalam satu bot.

Add to Discord Lihat Fitur ↓
Fitur

Semua yang kamu butuhkan

Dirancang untuk jadi teman ngobrol sekaligus asisten yang berguna di Discord.

🤖
AI Chat Multi-Provider
Didukung Groq (LLaMA 3.3 70B) dan OpenRouter (DeepSeek) dengan fallback otomatis. Respons cepat, pintar, dan ga pernah mati.
🔍
Web Search Real-time
Cukup mention "cari" atau "cariin" — Denia langsung search web dan kasih informasi terbaru. Didukung Serper (Google Search API).
🧠
Memory Per Channel
Denia ingat konteks percakapan di setiap channel. Ngobrol natural tanpa perlu ngulang cerita dari awal.
🎭
Persona System
Ganti karakter Denia sesuai mood. Setiap persona punya gaya ngobrol dan kepribadian yang berbeda.
📋
Joki Manager (DM Only)
Track akun joki kamu via DM. Tiap user punya list sendiri, aman dan private. Support sistem mingguan (Rabu ke Rabu).
⚡
Typing Simulation
Denia simulate typing sebelum reply — terasa lebih natural dan manusiawi saat ngobrol.
Commands

Slash Commands

Semua command tersedia via slash command — ketik / di Discord untuk lihat daftarnya.

/help
Lihat daftar command dan cara pakai bot
/status
Cek status bot, uptime, dan provider aktif
/stats
Statistik total pesan yang diproses
/forget
Hapus memory percakapan di channel ini
/persona [nama]
Ganti persona/karakter Denia
/debug
Info debug RAM & Node.js (owner only)
